# tweetsched-cron

Cron jobs service for the Scheduled Tweets service.

## Requirements:
 - Java 8 or higher
 - Maven 3.3.3 or higher

## How to build:
`mvn clean package`

## How to run locally:

`java -jar target/tweetsched-cron-0.1-SNAPSHOT.jar`
